## Как получить API-ключ
1. Откройте сайт [ExchangeRate-API](https://www.exchangerate-api.com/) для получения внешний API
2. Зарегистрируйтесь через кнопку "*Sign In*"
3. В поле ввода укажите свою действующую email-почту и нажмите на кнопку "*Get API Key*"
4. Подтвердите email, перейдите по ссылке и вы увидите ключ в "Your API key:": 
[Скриншот 20-06-2026.png](https://github.com/catsuper1020453-byte/Catsuper/blob/main/%D0%A1%D0%BA%D1%80%D0%B8%D0%BD%D1%88%D0%BE%D1%82%2020-06-2026.png)
5. Откройте файл "**main.py**" и замените строку: 
>API_KEY = "..."

## Описание программы
**Currency Converter - это GUI-приложение для конвертации валют в реальном времени.
Программа позволяет быстро переводить суммы между 160+ валютами, сохранять историю операций и экспортировать данные в удобных форматах.**
### Возможности:
- ✅ Конвертация 160+ валют в реальном времени
- ✅ Выбор валют (из и в)
- ✅ Выбор валют (из и в)
- ✅ Поле ввода суммы с валидацией (положительное число)
- ✅ Таблица истории с датой, временем и курсом
- ✅ Автосохранение истории в JSON
- ✅ Загрузка истории из JSON
- ✅ Экспорт в CSV
- ✅ Очистка истории
- ✅ Обработка ошибок (интернет, API, ввод)

## Инструкция по запуску
### Требования:
- Python 3.6 или выше

- Стабильное подключение к интернету (для получения актуальных курсов)

1. Откройте терминал или командную строку
2. Запустите приложение:


## Пример использования
### Конвертация USD в EUR:
1. Заполните поле "**Введите сумму:**" число
2. Выберите валюту "**Из:**" → USD
3. Выберите валюту "**В:**" → EUR
4. Нажмите кнопку "**Конвертировать**"
5. Все операции автоматически сохраняются в таблицу "История"
6. Дополнительные функции:\
Сохранить — экспорт истории в JSON-файл\
Загрузить — импорт истории из JSON-файла\
Очистить — удаление всей истории\
CSV — экспорт в формат Excel (CSV)
